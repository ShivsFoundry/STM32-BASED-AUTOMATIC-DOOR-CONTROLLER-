# Contributing

Thank you for your interest in contributing to the **STM32-Based Automatic Door Controller** project.

Contributions of all sizes are welcome, including bug fixes, documentation improvements, code enhancements, and new feature suggestions.

---

## How to Contribute

1. Fork this repository.
2. Create a new branch.

```bash
git checkout -b feature/your-feature-name
```

3. Make your changes.
4. Commit your work with a clear message.

```bash
git commit -m "Add: Brief description of changes"
```

5. Push the branch.

```bash
git push origin feature/your-feature-name
```

6. Open a Pull Request describing your changes.

---

## Contribution Guidelines

Please ensure that your contribution:

- Maintains the existing project structure.
- Includes clear and readable code.
- Preserves project functionality.
- Follows consistent formatting and comments.
- Updates documentation when required.

---

## Reporting Issues

If you encounter a bug or have a feature request:

- Check existing Issues first.
- Create a new Issue with a clear title.
- Provide steps to reproduce the problem.
- Include screenshots or logs when helpful.

---

## Code Style

Please follow these practices:

- Write clean and readable code.
- Use meaningful variable names.
- Add comments where appropriate.
- Keep formatting consistent.
- Avoid unnecessary code changes.

---

## Documentation

Documentation improvements are always appreciated.

Examples include:

- Correcting errors
- Improving explanations
- Adding setup instructions
- Enhancing project readability

---

## Code of Conduct

By participating in this project, you agree to communicate respectfully and collaborate in a positive and inclusive manner.

---

Thank you for helping improve this project and supporting the open-source community.

Happy Coding! 🚀
